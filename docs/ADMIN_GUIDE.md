# Admin Dashboard Guide

## Akses Admin Dashboard

### Quick Start
1. Buka website di browser
2. Tekan **Ctrl + Shift + A** untuk buka admin
3. Pilih section yang mau diedit
4. Klik tombol Preview untuk test
5. Klik Simpan Permanen untuk save

## 7 Editable Sections

### 1. Hero Section
- Banner image URL
- Main title
- Subtitle
- Button text & link

### 2. Marquee
- Scrolling text
- Color
- Speed

### 3. Bestseller
- 4 top products
- Product image, name, price

### 4. Collection
- Collection name
- Description
- Featured products

### 5. Our Picks
- 8 curated products
- Details & images

### 6. Stores
- Store locations
- Address & hours
- Contact info

### 7. Footer
- Brand info
- Social media
- Newsletter signup
- Copyright

## Tips
- Always Preview before saving
- Save data regularly
- Test on mobile devices
- Keep images optimized
- Update content seasonally

## Troubleshooting
- Clear browser cache if data lost
- Check console (F12) for errors
- Ensure images are accessible
- Verify localStorage is enabled
