📊 Dashboard Admin

Dashboard terdiri dari 3 bagian utama:

  • Sidebar Kiri (Menu) - Pilih section untuk diedit
  • Editor Panel (Tengah) - Form untuk input data
  • Preview Panel (Kanan) - Lihat perubahan real-time

Workflow Editing

  1. Pilih section di sidebar kiri
  2. Isi form di editor panel
  3. Klik "Preview" untuk lihat hasilnya
  4. Jika sudah OK, klik "Simpan Permanen"
  5. Data akan tersimpan di browser
✅ Penting: Selalu klik "Simpan Permanen" agar data tidak hilang saat refresh browser!

🏠 Hero Section

Section pertama yang dilihat pengunjung. Gunakan untuk showcase produk unggulan atau campaign terbaru.

Field yang Bisa Diedit

Field Deskripsi Contoh
Title Judul utama hero section "Welcome to NINETYNINE"
Subtitle Sub-title atau tagline "Fashion for Everyone"
Image URL Link gambar background "https://cdn.example.com/hero.jpg"
Button Text Teks tombol CTA "Shop Now" atau "Lihat Koleksi"

Tips Maksimalkan Hero Section

  • Gunakan gambar berkualitas tinggi (min 1200px width)
  • Judul max 50 karakter untuk mobile responsiveness
  • Subtitle buat ringkas dan menarik
  • Button text jangan lebih dari 20 karakter
  • Update setiap campaign atau season

📜 Marquee Section

Teks berjalan untuk announcement, promo, atau highlight special offers.

Field yang Bisa Diedit

Contoh Penggunaan

💡 Tips & Trik

Content Management

  • Update content minimal 1-2 minggu sekali
  • Keep copy & description SEO-friendly
  • Use consistent brand voice
  • Highlight seasonal atau trending items

Image Optimization

  • Compress images sebelum upload
  • Gunakan format JPEG untuk photos, PNG untuk graphics
  • Optimal size: 800-1200px width
  • File size max 500KB per image

Performance

  • Monitor page load speed
  • Cache images di CDN
  • Remove unused media files
  • Enable browser caching
⚠️ Backup Reminder: Backup localStorage data setiap bulan untuk mencegah data loss!

❓ Troubleshooting & FAQ

Q: Dashboard tidak muncul saat tekan Ctrl+Shift+A

A: Pastikan Anda berada di halaman utama (index.html), buka console (F12) untuk check error messages, atau refresh browser (Ctrl+R)

Q: Data hilang setelah close browser

A: Data disimpan di localStorage. Jangan clear browser cache. Jika terhapus, data akan kembali ke default.

Q: Gambar tidak loading di preview

A: Check image URL sudah benar dan file masih accessible. Verify di browser jika URL bisa dibuka.

Q: Edit tidak tersimpan

A: Pastikan klik "Simpan Permanen" bukan hanya "Preview". Check console untuk error messages.

Q: Website loading sangat lambat

A: Compress images lebih aggressive, gunakan CDN untuk images, minimize CSS/JS files, atau reduce jumlah products.

💡 Pro Tip: Backup data regularly dengan copy-paste dari browser console:
JSON.stringify(localStorage.getItem('ninetynine_admin_data'))
